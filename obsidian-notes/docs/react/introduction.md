# React 開發入門

React 是目前最受歡迎的前端 JavaScript 函式庫，用於建構用戶介面。

## 🎯 學習目標

- 理解 React 的核心概念
- 掌握組件化開發思維
- 學會狀態管理和生命週期
- 熟悉 React Hooks

## 📚 核心概念

### 1. 組件 (Components)
- 函數組件 vs 類別組件
- JSX 語法
- Props 傳遞
- 組件組合

### 2. 狀態管理
- useState Hook
- useEffect Hook
- Context API
- 狀態提升

### 3. 事件處理
- 事件綁定
- 表單處理
- 受控組件
- 非受控組件

### 4. 生態系統
- React Router (路由)
- State Management (Redux, Zustand)
- UI 函式庫 (Material-UI, Ant Design)
- 測試工具 (Jest, React Testing Library)

## 💡 最佳實踐

- 保持組件單一職責
- 合理使用 Hooks
- 避免不必要的重新渲染
- 適當的錯誤邊界處理

## 🛠️ 開發工具

- Create React App
- Next.js
- Vite
- React DevTools
