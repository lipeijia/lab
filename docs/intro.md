# 歡迎來到我的練習場 („• ֊ •„)੭

我是目前還在待業中的前端(笑)
從網頁設計轉職成為前端工程師也三年多了，其實有很多學習歷程沒有紀錄，全部都留在瀏覽器的頁籤上，
除了公司專案之外，沒有Side Project專案可以說嘴，更沒有貢獻過開源，
連自己的網站都沒有...實在有點慚愧 ( ;´ - `;)

決定建造自己的軍火庫，除了紀錄自己的學習筆記，
也開始學著寫 blog，分享想法！

雖然現在 AI 就是個人的超強家教，
